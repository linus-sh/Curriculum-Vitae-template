　最近携わったプロジェクトではWebアプリの実行環境などのインフラ構築を担当しました。実行環境には数あるクラウドサービスの中でもトップシェアを誇り、また単なる実行環境にとどまらず、CI/CDやサーバレス環境など現在のトレンドの技術を実現できることからAWSを選択しました。<br>
